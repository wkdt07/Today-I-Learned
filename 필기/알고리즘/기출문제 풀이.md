## 사각형 그리기

오른쪽 아래로 탐색해야 한다.
for문 순회할 때 어떻게 순회해야 하는지

```py
T = int(input())

for t in range(1,T+1):
    N = int(input())
    arr = [list(map(int,input().split)) for _ in range(N)]
    maxarea = 0
    cnt = 0
    for y in range(N):
        for x in range(N):
            cur = arr[y][x] #왼쪽 위의 좌표값

            #이제 오른쪽 아래로 탐색해야한다
            for ny in (y,N):
                for nx in (x,N)": # 시작 위치를 확인
                    if arr[ny][nx] == cur: # 동일한 값을 만나면 ---> 면적 계산
                        area = (ny-y+1)*(nx-x+1)
                        # 이 아래 코드가 가장 중요
                        if area > maxarea:
                            maxarea = area # 최대값이라면 갱신
                            cnt = 1
                        elif area == maxarea: # 같은 크기의 면적이 있으면
                            cnt+=1 # 누적

    print(f'#{t} {cnt}')
```

## 답안지 채점

10기 1차 im 기출

보너스를 체크해야한다

```py
T = int(input())

for t in range(1,T+1):
    N,M = map(int,input().split())
    ans = list(map(int,input().split()))
    max_v = float('-inf')
    min_v = float('inf')

    for _ in range(N):
        total = 0
        bonus = 1
        std_ans = list(map(int,input().split()))
        for j in range(M): # 문제의 개수
            if std_ans[j] == ans[j]:
                total += bonus
                bonus += 1 # 문제를 연속으로 맞출 수록 보너스 누적
            else:
                bonus = 1 # 못 맞출 경우 보너스 초기화

        max_v = max(total,max_v)
        min_v = min(total,min_v)
        result = max_v - min_v
    print(f'#{t} {result}')
```

## 중계기

올림 하는 것 때문에 정답률이 낮음

math 모듈의 ceil 쓰거나, 직접 짜라(int값과 비교해서 더 크면 int(x)+1)

```py

import math

T = int(input())
for t in range(1,T+1):
    N = int(input())
    area = [list(map(int,input().split())) for _ in range(N+1)]
    least_R = 0
    hi = hj = 0
    lst = []

    for i in range(N+1):
        for j in range(N+1):
            if area[i][j] == 1:
                lst.append([i,j])

            if area[i][j] == 2:
                hi,hj = i,j

    for l in lst:
        radius = ((l[0]-hi) ** 2 + (l[1]-hj) ** 2) ** 0.5
        least_R = max(radius,least_R)
        result = math.ceil(least_R)

    print(f'#{t} {result}')

```
